⚙️ 設定

Notion APIの設定を行います。

NotionのIntegration Tokenを入力してください（ntn_ または secret_ で始まる文字列）
タスク管理データベースのID（32文字の英数字）を入力してください

📖 セットアップ手順

  1. Notion Integrationsにアクセスし、「新しいインテグレーション」を作成
  2. 作成したIntegrationの「Internal Integration Token」（ntn_XXX または secret_XXX）をコピーし、上記の「Notion API Token」に入力
  3. タスク管理用Notionデータベースを開き、以下のプロパティを作成:
    • Name（タイトル）
    • 実行日（日付）
    • 完了（チェックボックス）
  4. データベースの「…」メニュー → 「接続を追加」→ 作成したIntegrationを選択
  5. データベースのURL（https://notion.so/xxxxx?v=yyyyy）から、Database ID（xxxxxの部分、32文字）をコピーし、上記の「Database ID」に入力
  6. 「保存」ボタンをクリック
